# Application to recognize images (AI)
REQUIREMENTS:
- Python 3.10
- Django 4.0
- Pillow
- numpy
- pandas
- tensorflow-hub
- tensorflow 2.8.0
- protobuf 3.20

## Installation:
```sh
pip install -r requirements.txt
```

## CREATE SUPER ADMIN USER:
```sh
python manage.py createsuperuser
```

## RUN:
```sh
python manage.py runserver
```
